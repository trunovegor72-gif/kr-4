Простое и удобное веб-приложение для конвертации валют, разработанное на React. Приложение использует фиксированные курсы валют и предоставляет интуитивно понятный интерфейс для быстрых расчетов.

✨ Возможности:

 Конвертация между валютами: USD, EUR, GBP, JPY, RUB

 Гибкий ввод сумм: Поддержка дробных чисел

 Быстрый обмен: Кнопка для мгновенной смены валют местами

 Прозрачные курсы: Отображение текущего курса конвертации

 Адаптивный дизайн: Оптимальное отображение на всех устройствах

 Мгновенные расчеты: Результаты обновляются в реальном времени
